# siu deps

> Adds or removes NPM dependencies to the specified packages

```bash
Usage: siu deps [options] <deps>

Add deps to target monorepo's package, e.g. add foo,foo@1.2.2,foo:D,foo@1.2.2:D

Options:

  -r, --rm               is remove deps from package
```
